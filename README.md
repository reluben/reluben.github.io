# Useful Links — Homepage

A simple [homepage](https://reluben.github.io) that puts useful links all in one place and reduces the need for bookmarks.

# Features 
- Local [disc golf notes](https://reluben.github.io/discgolf.html) 

## Links (in order)
1. Watching movies and TV shows
2. Watching anime
3. My (this) GitHub page
4. AI models
5. Timer — A timer for tracking project time (useful for billable work)  
6. [Word Games](./quickwordgames.html) — word games
7. [Alternatives](./alternatives.html) — fallback links if primary links are down


## Usage
- Clone the repo and open the main HTML file in your browser.
- Edit link targets and add or remove categories as needed.
